# Red Panda's Pocket Dungeon

_Red Panda's Pocket Dungeon_ is a simple open source dungeon crawler,
featuring Amber, the red panda adventurer.

The game is released for free as an open-source project
using the SFML Multimedia library and source code.

## Requirements

* `cmake`
* c++ compiler
* Editor

The project is supposed to work out of the box on most systems.
Linux systems might need to install the required libraries:

```bash
sudo apt  &&
sudo apt install \
   libxrandr-dev \
   libxcursor-dev \
   libxi-dev \
   libudev-dev \
   libfreetype-dev \
   libflac-dev \
   libvorbis-dev \
   libgl1-mesa-dev \
   libegl1-mesa-dev \
   libfreetype-dev
```
